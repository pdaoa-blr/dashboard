# Pavani Divine Dashboard 

A simple HTML, CSS, and JavaScript dashboard application for apartment community management.

[🌐 View Dashboard](https://pdaoa-blr.github.io/dashboard/)

## Features

- **Dashboard**: Overview with activity feed, expense summary donut chart, and committee members
- **Financials**: Monthly financial breakdown with income/expense tables
- **Facilities**: Status of community amenities (swimming pool, gym, party hall, etc.)
- **Projects**: Track ongoing and completed community projects
- **Documents**: Access to important documents, newsletters, and communication channels

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling
- `app.js` - JavaScript for rendering and interactivity
- `data.json` - All dashboard data (easily editable)

## Usage

1. Open `index.html` in a web browser
2. Or serve with any static file server:
   ```bash
   npx serve.
   ```

## Customization

Edit `data.json` to update:
- Committee members
- Financial data
- Facility statuses
- Project information
- Documents and links
- Newsletters

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
